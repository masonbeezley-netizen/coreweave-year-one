🔴 Code Red — Shipping Now
What's Coming Next from Glean
You've heard the noise around AI coworking tools. Here's what Glean is shipping right now that goes way beyond chat — built for teams exactly like yours.
🖥️

Computer Use

Glean is building a desktop app bridge that exposes your local terminal and filesystem directly to Assistant. Point it at a local repo and cowork on files — like Claude Code, but with all of Glean's enterprise context behind it. Imagine CoreWeave engineers debugging locally while Glean automatically pulls in the relevant Confluence runbook and Jira context. One conversation, full stack awareness.

🌐

Browser Use

Glean will be able to navigate the browser, click through apps, and take actions across the long tail of tools that don't have APIs or connectors. For CoreWeave with all your bespoke internal tooling, this closes the gap on stuff that no connector will ever cover.

🧩

MCP Everywhere

Glean's MCP server already works inside Claude Code, Cursor, Windsurf, Codex, VS Code, and JetBrains — but the bigger play is Glean as an MCP host too. Glean Agents can call out to any remote MCP server. Your knowledge flows into dev tools, AND external tools flow into Glean agents. Both directions.

🎨

Interactive HTML Apps in Canvas

This is the wow factor. Ask Glean to build something and it scaffolds a full interactive web app right in Canvas — dashboards, data explorers, calculators, project boards — all wired to enterprise data. Not a static doc, not a PDF — a real working mini-app you can share and embed. Your GTM team could say "build me a pipeline dashboard from Salesforce" and get a live interactive thing back.

🛠️ Proof point: This page was built using Glean

Mason built this entire interactive site in a single Glean conversation — it searched across Sigma, Salesforce, Slack, and Google Docs, pulled CoreWeave's real adoption data, and generated this HTML page as a live artifact. No code editor, no designer. That's the kind of coworking we're talking about.

Coming to Your Glean Experience

Agents in Slack DM Sidebar

Access Glean Assistant and your custom agents directly from the Slack sidebar — no context switching. Browse agents, pick a response mode (Fast or Thinking), and chat without leaving your workspace.

🟢 Beta — Join Now

Public Mode in Slack Channels

Glean proactively shares trusted answers in Slack channels based on your public knowledge corpus. Your Ask-IT bot could get even smarter with this one.

✅ Live

Agent Sandbox (AFS)

A cloud-based sandbox where Glean's agent engine can run code, analyze data, generate visualizations, and produce rich interactive dashboards — all grounded in your enterprise data.

In Development

Agent Governance (ADLC)

Enterprise-grade agent lifecycle management — versioning, eval sets, approval workflows, draft & publish, and full traceability for agents running in production.

Beta

Auto Mode & Agent Orchestration

Autonomous agents that plan, reason, adapt, and execute complex multi-step workflows. Plus agent-to-agent communication via Google's A2A protocol — a true agent mesh.

Connect Glean to any third-party MCP server. Your agents can now use tools from across the ecosystem — Asana, ClickUp, custom internal servers, and more.
